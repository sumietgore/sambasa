# Sambasa

Sambasa is a simple chat application built using Next.js, Tailwind CSS, Shadcn UI, Postgres, Prisma, and Better Auth.

The word Sambasa is derived from Sanskrit word Saṃbhāṣā (संभाषा) which means "conversation".
